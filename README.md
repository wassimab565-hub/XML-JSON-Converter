This project is a JavaFX application that converts data between XML and JSON formats.

---

## I. Project Structure
The project contains two versions:

### 1. XML_JSON_API
- Uses an external API (org.json)
- Conversion is handled using JSON and XML utility classes

### 2. XML_JSON_NO_API
- Does not use any external API
- Conversion is implemented manually using DOM and string processing

---

## II. Features
- XML to JSON conversion
- JSON to XML conversion
- Inverse direction switching
- Input validation and error handling
- JavaFX graphical user interface

---

## III. Technologies Used
- Java
- JavaFX
- XML DOM Parser
- org.json library (API version only)

---

## IV. Requirements
- Java 21+
- JavaFX SDK: https://openjfx.io
- org.json library: https://mvnrepository.com/artifact/org.json/json
